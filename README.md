# Snail-Kotlin

Download
--------
You can download a jar from GitHub's [releases page][1].

Or use Gradle:

```gradle
repositories {
  jcenter()
}

dependencies {
  compile 'com.compass.snail:snail:0.0.1'
}
```

Or Maven:

```xml
<dependency>
  <groupId>com.compass.snail</groupId>
  <artifactId>snail</artifactId>
  <version>0.0.1</version>
  <type>pom</type>
</dependency>
```
